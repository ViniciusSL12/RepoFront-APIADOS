# MelhorAqui - 3 telas (Front-end only)

Projeto Next.js com apenas front-end conforme referência enviada:
- Tela 1: Splash laranja com logo (5s -> tela 2)
- Tela 2: Apresentação (5s -> tela de escolha)
- Tela 3: "Eu sou..." com opções Morador / Administrador (idêntica à referência)

Instruções:
1. npm install
2. npm run dev
3. Abrir http://localhost:3000

Obs: As imagens fornecidas já estão em public/assets/.
